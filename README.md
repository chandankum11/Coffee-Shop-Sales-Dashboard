# Coffee-Shop-Sales-Dashboard
Interactive Excel dashboard analyzing 149k+ coffee shop sales records across multiple stores. Includes KPIs, hourly/weekday trends, top products, and category performance using Pivot Tables, Charts, and slicers. Converts raw data into clear, actionable business insights.
